# FloatChat - AI-Powered Conversational Analytics for ARGO Ocean Data

An intelligent chatbot system powered by AI/ML technologies for analyzing and querying ARGO ocean data through natural language conversations.

## Tech Stack
- **LLM:** Ollama
- **Vector Database:** ChromaDB
- **Embeddings:** HuggingFace
- **Database:** PostgreSQL
- **RAG Pipeline:** Custom implementation
